# 📊 Система мониторинга цен (Учебная версия)

Автоматическая система отслеживания изменения цен на товары с использованием учебных сайтов для тестирования.

## 🎯 Учебные сайты

- **books.toscrape.com** - демо-магазин книг с реальными ценами
- **quotes.toscrape.com** - сайт с цитатами для тестирования парсинга
- **httpbin.org** - тестовый HTTP сервер

## 📦 Установка

1. Клонируйте репозиторий
2. Установите зависимости:
   ```bash
   pip install -r requirements.txt
3. Создайте конфигурационный файл:
    ```bash
   python main.py --once  # автоматически создаст config.json
