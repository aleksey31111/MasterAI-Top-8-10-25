🎯 Для scraper.py (подсказка по селекторам)
python
# Для Books to Scrape
def parse_book_price(soup):
    price_element = soup.find('p', class_='price_color')
    if price_element:
        price_text = price_element.text.strip().replace('£', '')
        return float(price_text)
    return None

# Для WebScraper Demo
def parse_laptop_price(soup):
    price_element = soup.find('h4', class_='price')
    if price_element:
        price_text = price_element.text.strip().replace('$', '')
        return float(price_text)
    return None
⚠️ Важные замечания
Эти сайты специально созданы для тестирования — они не заблокируют ваш IP

Цены меняются редко — для тестирования уведомлений можно искусственно занизить пороги

Books to Scrape содержит 1000 книг с пагинацией — отлично для тестирования

Не используйте реальные маркетплейсы для тестов — Ozon, Wildberries, Amazon могут заблокировать за частые запросы

🚀 Как быстро протестировать
Скопируйте пример config.json выше

Замените email-настройки на свои (для Gmail нужен пароль приложения)

Запустите python main.py

Проверьте, что цены успешно парсятся

При необходимости измените пороги, чтобы протестировать уведомления

Эти сайты гарантированно подходят для отладки и позволят вам проверить все компоненты вашей системы мониторинга цен!
